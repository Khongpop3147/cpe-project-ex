ในบทนี้จะกล่าวถึงผลลัพธ์ของการพัฒนาแพลตฟอร์ม Elder Store ตลอดจนการประเมินผลประสิทธิภาพการทำงานและความพึงพอใจจากผู้ใช้งานจริง ซึ่งประกอบไปด้วยการประเมินจากภาคธุรกิจผู้เชี่ยวชาญ และกลุ่มเป้าหมายหลักคือผู้สูงอายุ

\section{\ifenglish System Implementation Results\else ผลการพัฒนาระบบและหน้าจอการใช้งาน (System Interfaces)\fi}
จากการรวบรวมความต้องการและขั้นตอนการออกแบบสถาปัตยกรรมระบบ คณะผู้จัดทำได้ดำเนินการพัฒนาเว็บไซต์ Elder Store จนเสร็จสมบูรณ์ โดยประยุกต์ใช้หลักการพัฒนาอย่างคล่องตัว (Agile Methodology) \cite{beck2001} ระบบถูกแบ่งส่วนการแสดงผลออกเป็น 2 ส่วนหลัก ได้แก่ ส่วนติดต่อผู้ใช้งานทั่วไป (User Interfaces) \cite{react2023} และส่วนระบบจัดการหลังบ้าน (Admin Dashboard) ซึ่งมีผลลัพธ์หน้าจอการทำงานดังต่อไปนี้

% =========================================================
% 4.1 ส่วนผู้ใช้งานทั่วไป (User Interfaces)

\subsection{หน้าจอส่วนติดต่อผู้ใช้งานทั่วไป (User Interfaces)}
ส่วนนี้คือหน้าเว็บที่พัฒนามุ่งเน้นสำหรับกลุ่มผู้สูงอายุและผู้บริโภคทั่วไป (Frontend System) โดยยึดหลักการออกแบบที่ไม่สร้างภาระทางความคิด (Don’t Make Me Think) \cite{krug2000} ประกอบด้วยหน้าจอต่างๆ ดังนี้

\subsubsection{หน้าจอหลัก (Home Page)}
หน้าด่านแรกของระบบ ประกอบด้วยโปรโมชัน แบนเนอร์สินค้าแนะนำ (Recommended) และสินค้ายอดนิยม (Popular) ที่ถูกจัดวางอย่างเป็นระเบียบตามหลักสถาปัตยกรรมสารสนเทศ เพื่อให้ผู้สูงอายุนำทางได้ง่ายและไม่หลงทาง \cite{shneiderman2010}
\begin{figure}[h!]
    \centering
    % \includegraphics[width=0.9\textwidth]{รูป_home.png}
    \caption{การแสดงผลหน้าจอหลัก (index.tsx)}
\end{figure}

\subsubsection{หน้าจอรายการสินค้าทั้งหมด (All Products Page)}
หน้าจอแสดงรายการแคตตาล็อกสินค้า (all-products.tsx) พร้อมระบบคัดกรองตามหมวดหมู่ (Categories) ที่ใช้ตัวอักษรขนาดใหญ่และจุดที่มีความเปรียบต่างสีสูงตามมาตรฐานสากล \cite{wcag2023}

\end{figure}

\subsubsection{หน้าจอรายละเอียดสินค้า (Product Detail Page)}
หน้านำเสนอข้อมูลสินค้าเฉพาะรายการ ระบุราคา คุณสมบัติอย่างชัดเจน โปร่งใส \cite{nielsen2000} และกำหนดจุดวางปุ่มเพิ่มลงตะกร้าสินค้าให้มีขนาดเป้าหมายรับสัมผัส (Touch Target) ที่ปลอดภัยนิ้ว \cite{mobileux2021}

\end{figure}

หน้าต่างสำหรับยืนยันตัวตนก่อนเข้าใช้งาน หรือสร้างบัญชีผู้ใช้งานใหม่ ออกแบบให้ใช้ฟิลด์ข้อมูลหน้าฟอร์มที่จำเป็นและเข้ารหัสปลอดภัยตามหลักวิศวกรรมข้อมูล \cite{pressman2014} เพื่อลดภาวะวิตกกังวล

\subsection{หน้าจอระบบจัดการหลังบ้าน (Admin Ecosystem)}
ระบบนี้เป็นพื้นที่ศูนย์บัญชาการควบคุม (Control Center) สำหรับผู้ดูแลเว็บไซต์และผู้ประกอบการร้านค้า ที่ถูกปกป้องด้วยระบบยืนยันวงเวลายืนยันตนระดับ Admin โดยเฉพาะ เพื่อใช้ดูความเคลื่อนไหวข้อมูลหลังบ้านแบบเรียลไทม์ \cite{nextjs2023}

\subsubsection{หน้าจอเข้าสู่ระบบผู้ดูแลหลังบ้าน (Admin Login)}
ประตูด่านแรกของการเข้าสู่ระบบหลังบ้าน ซึ่งแยกสายความปลอดภัยและการเข้ารหัสสิทธิ์ (Rights Management) ออกจากผู้ใช้งานทั่วไปโดยเด็ดขาด
