\chapter{\ifenglish Introduction\else บทนำ\fi}

\section{\ifenglish Project rationale\else ที่มาของโครงงาน\fi}
โครงการนี้เกิดจากการเล็งเห็นว่า ผู้สูงอายุในสังคมยังขาดแพลตฟอร์มที่ช่วยให้เข้าถึงสินค้าและบริการได้ง่าย 
โดยเฉพาะในยุคที่เทคโนโลยีมีบทบาทสำคัญในชีวิตประจำวัน 
ดังนั้นจึงได้พัฒนาเว็บไซต์ Elder Store ที่เน้นการออกแบบให้ใช้งานง่าย 
และตอบโจทย์ความต้องการของผู้ใช้งาน
เพื่อเพิ่มโอกาสในการเข้าถึงสินค้าและบริการที่จำเป็นในชีวิตประจำวัน

\section{\ifenglish Objectives\else วัตถุประสงค์ของโครงงาน\fi}
\begin{enumerate}
    \item พัฒนาเว็บไซต์ Elder Store ที่ใช้งานง่ายสำหรับผู้สูงอายุ
    \item ออกแบบระบบแนะนำสินค้าที่เหมาะสมกับความต้องการของผู้ใช้งาน
    \item สร้างแพลตฟอร์มที่ช่วยเพิ่มคุณภาพชีวิตของผู้สูงอายุผ่านการใช้งานเทคโนโลยี
    \item ส่งเสริมการเข้าถึงสินค้าและบริการที่จำเป็นในชีวิตประจำวัน
\end{enumerate}

\section{\ifenglish Project scope\else ขอบเขตของโครงงาน\fi}

\subsection{\ifenglish Hardware scope\else ขอบเขตด้านฮาร์ดแวร์\fi}
ผู้ใช้งานสามารถเข้าถึงระบบผ่านอุปกรณ์ที่รองรับอินเทอร์เน็ต เช่น คอมพิวเตอร์ สมาร์ตโฟน 
และแท็บเล็ต โดยไม่จำเป็นต้องติดตั้งอุปกรณ์เพิ่มเติม

\subsection{\ifenglish Software scope\else ขอบเขตด้านซอฟต์แวร์\fi}
โครงการครอบคลุมการพัฒนาเว็บไซต์สำหรับผู้สูงอายุ ระบบแนะนำสินค้า 
และการออกแบบที่คำนึงถึงการใช้งานที่สะดวกและปลอดภัย

\section{\ifenglish Expected outcomes\else ประโยชน์ที่ได้รับ\fi}
\begin{itemize}
    \item ผู้สูงอายุสามารถเข้าถึงสินค้าและบริการได้สะดวกขึ้น
    \item ระบบแนะนำสินค้าช่วยเพิ่มประสบการณ์การใช้งานที่ดีแก่ผู้ใช้
    \item ส่งเสริมคุณภาพชีวิตของผู้สูงอายุผ่านการใช้งานเทคโนโลยี
    \item สร้างแพลตฟอร์มที่ช่วยลดความซับซ้อนในการใช้งานเทคโนโลยี
\end{itemize}

\section{\ifenglish Technology and tools\else เทคโนโลยีและเครื่องมือที่ใช้\fi}

\subsection{\ifenglish Hardware technology\else เทคโนโลยีด้านฮาร์ดแวร์\fi}
เซิร์ฟเวอร์สำหรับให้บริการเว็บไซต์ และอุปกรณ์ของผู้ใช้ เช่น คอมพิวเตอร์และสมาร์ตโฟน

\subsection{\ifenglish Software technology\else เทคโนโลยีด้านซอฟต์แวร์\fi}
ซอฟต์แวร์และเครื่องมือที่ใช้ในการพัฒนาโครงการนี้ประกอบด้วย:
\begin{itemize}
    \item \textbf{Frontend:} React.js ร่วมกับ Tailwind CSS 
    เพื่อสร้างส่วนติดต่อผู้ใช้ที่เรียบง่ายและเหมาะสมกับผู้สูงอายุ
    \item \textbf{Backend:} Next.js (Node.js + TypeScript) 
    รองรับทั้ง Server-side Rendering (SSR) และ API สำหรับการติดต่อกับฐานข้อมูล
    \item \textbf{Database:} PostgreSQL 
    ใช้จัดเก็บข้อมูลผู้ใช้ สินค้า และคำสั่งซื้อ
    \item \textbf{Authentication:} OAuth 2.0 และ JWT 
    เพื่อจัดการสิทธิ์การเข้าถึงและความปลอดภัยของผู้ใช้งาน
    \item \textbf{Accessibility:} การออกแบบตามมาตรฐาน WCAG 2.1 
    เพื่อรองรับผู้ใช้งานทุกกลุ่ม รวมถึงผู้สูงอายุ
\end{itemize}

\section{\ifenglish Roles and responsibilities\else บทบาทและความรับผิดชอบ\fi}
ในการทำงานได้มีการแบ่งบทบาทของนักศึกษาออกเป็นกลุ่ม ได้แก่:
\begin{itemize}
    \item ฝ่ายวิเคราะห์และเก็บข้อมูลความต้องการผู้ใช้
    \item ฝ่ายออกแบบระบบและ UI/UX
    \item ฝ่ายพัฒนาและทดสอบระบบ (Frontend/Backend)
    \item ฝ่ายจัดทำเอกสารและนำเสนอผล

\section{\ifenglish Team Roles\else บทบาทของทีม\fi}
ทีมของเราประกอบด้วยสมาชิก 3 คน โดยมีการแบ่งบทบาทดังนี้:
\begin{itemize}
    \item \textbf{Backend Developer:} รับผิดชอบการพัฒนาและดูแลระบบหลังบ้าน รวมถึงการจัดการฐานข้อมูลและ API
    \item \textbf{Frontend Developer (Web):} ดูแลการพัฒนาและออกแบบส่วนติดต่อผู้ใช้สำหรับเว็บไซต์
    \item \textbf{Frontend Developer (Mobile):} รับผิดชอบการพัฒนาและออกแบบส่วนติดต่อผู้ใช้สำหรับแอปพลิเคชันมือถือ
\end{itemize}
