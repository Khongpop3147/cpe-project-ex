\begin{abstract}
\par
โครงการพัฒนาแพลตฟอร์มพาณิชย์อิเล็กทรอนิกส์ "Elder Store" มีวัตถุประสงค์เพื่อลดช่องว่างความเหลื่อมล้ำทางดิจิทัล (Digital Divide) และสร้างสภาพแวดล้อมการซื้อขายออนไลน์ที่เป็นมิตรต่อกลุ่มผู้สูงอายุ เนื่องจากแพลตฟอร์มในปัจจุบันมีความซับซ้อนและก่อให้เกิดอุปสรรคต่อข้อจำกัดทางสรีรวิทยาของผู้ใช้งานกลุ่มนี้ โครงงานนี้จึงได้ประยุกต์ใช้หลักการอารยสถาปัตย์ (Inclusive Design) และมาตรฐานการเข้าถึงโครงสร้างเนื้อหาเว็บไซต์ (WCAG 2.1) มาเป็นแนวทางหลักในการออกแบบประสบการณ์ผู้ใช้งาน (UX/UI) เพื่อขจัดสิ่งรบกวนตา ลดความซับซ้อนในระบบนำทาง และขยายขนาดเป้าหมายการสัมผัสให้ปลอดภัยและเหมาะสม

\par
ระบบโครงสร้างสถาปัตยกรรมถูกพัฒนาขึ้นโดยใช้โมเดิร์นเว็บเทคโนโลยี ได้แก่ React.js และ Next.js สำหรับระบบส่วนหน้า (Frontend) ที่ประมวลผลตอบสนองได้อย่างรวดเร็ว ควบคู่กับการใช้ฐานข้อมูลเชิงสัมพันธ์ PostgreSQL ในการบริหารจัดการคลังสินค้าและระบบหลังบ้าน (Admin Dashboard) นอกจากนี้ยังได้พัฒนาระบบคัดกรองแนะนำสินค้าอัจฉริยะที่สอดคล้องกับความต้องการของผู้สูงอายุ พร้อมผสานกระบวนการชำระเงินแบบลดขั้นตอน (Single-step Checkout) เพื่อให้เกิดความปลอดภัยรัดกุมสูงสุด

\par
ผลจากการทดสอบประสิทธิภาพระบบร่วมกับผู้เชี่ยวชาญจากบริษัท Iconnex Thailand และกลุ่มตัวอย่างผู้สูงอายุใช้งานจริง พบว่า แพลตฟอร์มสามารถตอบสนองการทำธุรกรรมได้อย่างมีเสถียรภาพ ผู้ใช้งานมีความพึงพอใจและสามารถดำเนินการสั่งซื้อสินค้าได้ด้วยตนเองอย่างอิสระ โครงงานนี้จึงถือเป็นนวัตกรรมที่ส่งเสริมคุณภาพชีวิต เติมเต็มความภาคภูมิใจ และสนับสนุนความก้าวหน้าของสังคมผู้สูงอายุ (Aging Society) อย่างเป็นรูปธรรม
\end{abstract}

\begin{abstract}
\par
The development of the "Elder Store" e-commerce platform aims to bridge the digital divide and establish an elderly-friendly online shopping environment. Recognizing the physiological and cognitive barriers posed by the complexity of contemporary platforms, this project applied Inclusive Design principles and the Web Content Accessibility Guidelines (WCAG 2.1) to engineer a user interface (UI/UX) that eliminates visual distractions, mitigates cognitive load, and optimizes touch targets for elderly users.

\par
The system architecture was constructed using modern web technologies, specifically React.js and Next.js for a highly responsive frontend, coupled with a PostgreSQL relational database to manage backend inventory and secure transaction histories seamlessly. Additionally, a tailored product recommendation algorithm was implemented to align with the elderly's behavioral preferences, alongside a streamlined single-step checkout process to ensure maximum security and navigational convenience.

\par
System evaluations, conducted in collaboration with technical experts from Iconnex Thailand and elderly target users, demonstrated that the platform operates with high efficiency and structural reliability. Users were able to independently navigate, interact, and complete their purchases safely. Consequently, this project serves as a significant technological advancement in enhancing the quality of life, fostering digital self-reliance, and supporting the infrastructure of an increasingly aging society.
\end{abstract}
